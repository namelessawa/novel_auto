# 无限小说生成系统 - 前端界面

这是一个基于Node.js和Express的图形化界面，用于操作无限小说生成系统。

## 功能特性

- 创建新小说：输入主题创建新的小说系列
- 续写小说：选择已有主题进行续写
- 实时输出：查看系统运行的实时输出
- 章节管理：浏览和查看已生成的章节
- 进程控制：启动和停止生成进程

## 安装和运行

### 前提条件

- Node.js (v14或更高版本)
- Python 3.x (已安装并配置好无限小说生成系统)

### 安装步骤

1. 确保您已在 `E:\pythonproject\novel_auto` 目录下安装了无限小说生成系统
2. 进入前端目录：
   ```bash
   cd frontend
   ```
3. 安装依赖：
   ```bash
   npm install
   ```
4. 启动服务器：
   ```bash
   npm start
   ```
   或者直接运行：
   ```bash
   node server.js
   ```

### Windows快捷启动

您可以使用提供的批处理文件启动服务器：
```bash
start.bat
```

## 使用说明

1. 打开浏览器访问 `http://localhost:8080`
2. 在"创建新小说"区域输入主题并点击"创建小说"
3. 在"续写已有小说"区域选择主题和模式后点击"开始续写"
4. 在"系统输出"区域查看实时运行状态
5. 在"可用主题"区域查看所有已创建的主题
6. 点击"查看章节"按钮浏览特定主题的章节内容

## 端口配置

默认端口为3000，如需修改，请编辑 `server.js` 文件中的 `PORT` 变量。

## 停止服务器

要停止服务器，请在终端中按 `Ctrl+C`。

## 注意事项

- 确保Python环境已正确配置，并且API密钥已设置
- 服务器运行期间会占用系统资源，请在不需要时及时关闭
- 如遇问题，请检查控制台输出以获取错误信息
